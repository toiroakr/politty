# db-query

データベースクエリの実行（ライフサイクルフックのデモ）

## Usage

```
db-query [options]
```

## Options

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--database <DATABASE>` | `-d` | データベース接続文字列 | - |
| `--query <QUERY>` | `-q` | SQLクエリ | - |
| `--simulate_error` | `-e` | エラーをシミュレート | `false` |
