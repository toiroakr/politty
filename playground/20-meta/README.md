# greet-meta

Test for Zod .meta() support

## Usage

```
greet-meta [options] <name>
```

## Arguments

| Argument | Description              | Required |
| -------- | ------------------------ | -------- |
| `name`   | Name to greet (via meta) | Yes      |

## Options

| Option                  | Alias | Description                | Default   |
| ----------------------- | ----- | -------------------------- | --------- |
| `--greeting <GREETING>` | `-g`  | Greeting phrase (via meta) | `"Hello"` |
