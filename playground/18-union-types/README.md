# auth-demo

Demo of union help with auth methods

**Usage**

```
auth-demo [options]
```

**Options**

| Option                  | Alias | Description | Required | Default |
| ----------------------- | ----- | ----------- | -------- | ------- |
| `--token <TOKEN>`       | -     | API Token   | Yes      | -       |
| `--username <USERNAME>` | -     | Username    | Yes      | -       |
| `--password <PASSWORD>` | -     | Password    | Yes      | -       |
