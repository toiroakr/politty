# auth-demo

Demo of union help with auth methods

## Usage

```
auth-demo [options]
```

## Options

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--token <TOKEN>` | - | API Token | - |
| `--username <USERNAME>` | - | Username | - |
| `--password <PASSWORD>` | - | Password | - |
