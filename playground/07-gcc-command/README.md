# gcc

Cコンパイラ（gccコマンド風）

## Usage

```
gcc [options] <sources>
```

## Arguments

| Argument | Description | Required |
|----------|-------------|----------|
| `sources` | ソースファイル | Yes |

## Options

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--output <OUTPUT>` | `-o` | 出力ファイル名 | - |
| `--optimize` | `-O` | 最適化を有効にする | `false` |
