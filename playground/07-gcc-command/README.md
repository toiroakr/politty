# gcc

C compiler (gcc command style)

**Usage**

```
gcc [options] <sources>
```

**Arguments**

| Argument  | Description  | Required |
| --------- | ------------ | -------- |
| `sources` | Source files | Yes      |

**Options**

| Option              | Alias | Description         | Default |
| ------------------- | ----- | ------------------- | ------- |
| `--output <OUTPUT>` | `-o`  | Output filename     | -       |
| `--optimize`        | `-O`  | Enable optimization | `false` |
