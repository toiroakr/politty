# greet

A CLI tool that displays greetings

**Usage**

```
greet [options] <name>
```

**Arguments**

| Argument | Description           | Required |
| -------- | --------------------- | -------- |
| `name`   | Name of the recipient | Yes      |

**Options**

| Option                  | Alias | Description      | Default   |
| ----------------------- | ----- | ---------------- | --------- |
| `--greeting <GREETING>` | `-g`  | Greeting phrase  | `"Hello"` |
| `--loud`                | `-l`  | Output uppercase | `false`   |
