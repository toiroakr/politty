# server

Server configuration example (type coercion and validation)

**Usage**

```
server [options]
```

**Options**

| Option            | Alias | Description           | Required | Default       |
| ----------------- | ----- | --------------------- | -------- | ------------- |
| `--port <PORT>`   | `-p`  | Port number (1-65535) | Yes      | -             |
| `--count <COUNT>` | `-n`  | Repeat count          | No       | `1`           |
| `--host <HOST>`   | `-h`  | Hostname              | No       | `"localhost"` |
