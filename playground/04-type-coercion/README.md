# server

サーバー設定の例（型変換とバリデーション）

## Usage

```
server [options]
```

## Options

| Option            | Alias | Description          | Default       |
| ----------------- | ----- | -------------------- | ------------- |
| `--port <PORT>`   | `-p`  | ポート番号 (1-65535) | -             |
| `--count <COUNT>` | `-n`  | 繰り返し回数         | `1`           |
| `--host <HOST>`   | `-h`  | ホスト名             | `"localhost"` |
