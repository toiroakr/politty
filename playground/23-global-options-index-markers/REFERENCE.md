# project-cli

Project management CLI demonstrating docs markers

## Global Options

These options are shared across `build` and `deploy` commands.

<!-- politty:global-options:start -->

| Option        | Alias | Description           | Default         |
| ------------- | ----- | --------------------- | --------------- |
| `--verbose`   | `-v`  | Enable verbose output | `false`         |
| `--env <ENV>` | `-e`  | Target environment    | `"development"` |

<!-- politty:global-options:end -->
