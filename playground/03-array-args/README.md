# process-files

Process multiple files

**Usage**

```
process-files [options]
```

**Options**

| Option            | Alias | Description                         | Default |
| ----------------- | ----- | ----------------------------------- | ------- |
| `--files <FILES>` | `-f`  | Files to process (multiple allowed) | -       |
| `--verbose`       | `-v`  | Verbose output                      | `false` |
