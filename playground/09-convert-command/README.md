# convert

ファイル形式を変換する（オプションpositionalの例）

**Usage**

```
convert [options] <input> [output]
```

**Arguments**

| Argument | Description                      | Required |
| -------- | -------------------------------- | -------- |
| `input`  | 入力ファイル                     | Yes      |
| `output` | 出力ファイル（省略時は標準出力） | No       |

**Options**

| Option              | Alias | Description | Default  |
| ------------------- | ----- | ----------- | -------- |
| `--format <FORMAT>` | `-f`  | 出力形式    | `"json"` |
