# build

<!-- politty:command::heading:end -->

<!-- politty:command::description:start -->

Build with cache and color toggles using custom negation names

<!-- politty:command::description:end -->

<!-- politty:command::usage:start -->

**Usage**

```
build [options]
```

<!-- politty:command::usage:end -->

<!-- politty:command::options:start -->

**Options**

| Option                        | Alias | Description              | Required | Default |
| ----------------------------- | ----- | ------------------------ | -------- | ------- |
| `--cache` / `--disable-cache` | -     | Use the build cache      | No       | `true`  |
| `--color`                     | -     | Colorize output          | No       | `true`  |
| `--monochrome`                | -     | Disable colorized output | No       | -       |

<!-- politty:command::options:end -->
