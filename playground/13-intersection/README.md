# process

ファイルを処理（intersectionの例）

## Usage

```
process [options] <input>
```

## Arguments

| Argument | Description  | Required |
| -------- | ------------ | -------- |
| `input`  | 入力ファイル | Yes      |

## Options

| Option              | Alias | Description  | Default |
| ------------------- | ----- | ------------ | ------- |
| `--verbose`         | `-v`  | 詳細出力     | `false` |
| `--config <CONFIG>` | `-c`  | 設定ファイル | -       |
| `--quiet`           | `-q`  | 出力を抑制   | `false` |
| `--output <OUTPUT>` | `-o`  | 出力ファイル | -       |
