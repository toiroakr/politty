# cat

ファイルの内容を表示する（catコマンド風）

## Usage

```
cat [options] <files>
```

## Arguments

| Argument | Description | Required |
|----------|-------------|----------|
| `files` | 表示するファイル | Yes |

## Options

| Option | Alias | Description | Default |
|--------|-------|-------------|---------|
| `--number` | `-n` | 行番号を表示 | `false` |
| `--show-ends` | `-E` | 行末に$を表示 | `false` |
