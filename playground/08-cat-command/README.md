# cat

Display file contents (cat command style)

**Usage**

```
cat [options] <files>
```

**Arguments**

| Argument | Description      | Required |
| -------- | ---------------- | -------- |
| `files`  | Files to display | Yes      |

**Options**

| Option        | Alias | Description            | Default |
| ------------- | ----- | ---------------------- | ------- |
| `--number`    | `-n`  | Show line numbers      | `false` |
| `--show-ends` | `-E`  | Show $ at end of lines | `false` |
