## build

Build the project

**Usage**

```
my-app build [options]
```

**Options**

| Option              | Alias | Description      | Required | Default  |
| ------------------- | ----- | ---------------- | -------- | -------- |
| `--output <OUTPUT>` | `-o`  | Output directory | No       | `"dist"` |

See [Global Options](README.md#global-options) for options available to all commands.

## deploy

Deploy the project

**Usage**

```
my-app deploy [options]
```

**Options**

| Option        | Alias | Description        | Required | Default        |
| ------------- | ----- | ------------------ | -------- | -------------- |
| `--env <ENV>` | `-e`  | Target environment | No       | `"production"` |

See [Global Options](README.md#global-options) for options available to all commands.
